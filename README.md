# IBank Testing

[![Java CI with Gradle](https://github.com/iraczar/ibank-testing/actions/workflows/gradle.yml/badge.svg)](https://github.com/твой-username/ibank-testing/actions/workflows/gradle.yml)

Автотесты для интернет-банка.

## Описание
Проект содержит автотесты для проверки функционала входа в систему интернет-банка.

## Технологии
- Java 11
- Selenide
- REST-assured
- JUnit 5
- Gradle

## Оценка времени тестирования

- Время, затраченное на ручное тестирование (минут): 35
- Время, затраченное на автоматизацию (минут): 150
